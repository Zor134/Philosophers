## Description

The goal of Philisophers project is to learn how to use threads while avoiding data race.

## Instructions

To compile the project use :
```Make```

To execute the program use :
```./philo <number_of_philosophers> <time_to_die> <time_to_eat> <time_to_sleep> [number_of_times_each_philosopher_must_eat]```
number_of_times_each_philosopher_must_eat is not required to execute.
